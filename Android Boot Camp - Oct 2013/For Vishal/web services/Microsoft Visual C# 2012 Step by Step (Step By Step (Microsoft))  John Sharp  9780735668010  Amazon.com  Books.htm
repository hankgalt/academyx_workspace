About the Author

John Sharp is the author of Microsoft Windows Communication Foundation Step by Step and Microsoft Visual C# 2010 Step by Step. An expert on developing applications with the Microsoft .NET Framework and interoperability issues, John has produced numerous tutorials, white papers, and presentations on distributed systems, web services, and the C# language.

More About the Author

› Visit Amazon's John Sharp Page

Biography

John Sharp gained an honors degree in Computing from Imperial College, London. He has been developing software and writing training courses, guides, and books for over 25 years. John has experience in a wide range of technologies, from database systems and UNIX through to C, C++ and C# applications for the .NET Framework, together with Java and JavaScript development. He has authored several books for Microsoft Press, including six editions of C# Step By Step, two editions of Windows Communication Foundation Step By Step, and the J# Core Reference.

John also writes for the Patterns and Practices group within Microsoft, and has helped to develop several guides covering a variety of areas, including Windows Azure, software development, and data access.

Excellent primer on C# 2012 for Win 7 and Win 8
By Jonathan C. Cauthorn on February 9, 2013
Format: Paperback
I've been programming in other languages so I have a good base to start from but this book does an excellent job of introducing me to C#. It is very easy to follow, they build on examples well, and introduce the object-oriented concepts well. I used to do a lot of simple programming 'glue' to make things work but this book broadens my understanding considerably.
It's a great way to learn C#. It covers programming issues for both the Win 7 and Win 8 bi-polar mode (desktop vs full-screen app). It also gives a nice introduction to Microsoft Visual Studio 2012 so that you learn the basics of debugging and using the editors and environment which can be very intimidating at first (I know it was for me!).
It covers a general topic in each chapter like methods, scope, objects and gives a real simple easy to understand definition and real world explanation of the concepts. This makes it very easy to follow.
I highly recommend this book.

Well organized, concise, content rich
By J. Kerr on June 5, 2013

I purchased this to teach myself Visual C#. I have developed using many languages, but never C.

My 2 cents.

If new to C# avoid this book
By Boris Bahes on February 15, 2013
Format: Paperback
Even reading C# 2010 edition I had problems understanding what author wanted to explain in most chapters in Part 1 and Part 2.
The examples are overly complicated for beginner. The concepts of C# are not that hard, but author examples make it hard to understand.
You are guided through examples in chapters with precise directions what you have to type (you are never challenged with any question which would force you to think) in code and with all examples you are left to break down application logic to understand what application really does and what author wanted to explain.

I have found that this "symptom" is to be found in many books for beginners. The author assumes what you don't know and forces using complete application project as a mean to explain concept. And as I said, you are left to break down application logic on your own to understand simple concept of chapter.

Also, by using this "complete application project" this book is full of notes like "You will learn this in Chapter XX".

My suggestion, for beginners is, first read book for professionals, and then after grasping C# concepts, return to this "book for beginners" and play with examples.

Also, you can not contact author directly. Instead you are left with Microsoft Press support to do this for you.

Regarding 2012 edition, the author complicates things even more by providing examples for Windows Store applications and Windows Desktop applications for same concept. The book is now harder to read thanks to smaller fonts.
